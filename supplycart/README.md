# supplycart

## Project setup
To run the web app, 
<ol>
  <li> At your terminal, enter:
    
```
npm install
```
    
  </li>
  <li> Then enter:
    
```
npm run serve
```
    
  </li>
  <li> Lastly, in your preferred web browser, go to http://localhost:8080/ or http://192.168.0.170:8080/, or the route that your terminal specifies
